Project 1:

In class we have been working on creating a virtual world environment using Minecraft. Our goal was to build a arena that we could drop players into, and would fight to the death like in hunger games. Together we do have a lot of experience in Minecraft, but not in the ‘building’ part of it so we was centred more on the buildings and the surrounding than in the more technical side (although we did touch on that a bit to make some things we wanted in the arena).

Some of our objectives

To build an enclosed area that we could drop players in. Make the area inside look good, while eliminating some unnecessary things inside eg, we put a layer of blue glass a block underneath the water so players couldn’t drown anywhere in the map and covering caves and such. Have different ‘zones’ as to not feel repetitive in the environment. To use as little commands as possible while the arena is in use.

Results / Disscusion

We also did a bit of ‘terra-forming’ (where you edit natural environments to make them look better) to make the arena unescapable and just make the overall area inside look a lot better. We also added some fun little secrets, that will remain a secret for now, to spice up the game and make it different if the secrets are discovered.

We had a fair number of problems when creating this: the world and game kept deleting itself every time we closed it, after the first deletion Mr Long was away (don’t blame you) and we had no copy of the game to use so we were delayed greatly, this is using an old version of the game so we didn’t have all of the options we would normally have when using Minecraft. But although having some problems, we tried our best to overcome them, and we are happy with what we have created. It could have been better, and we are proud of what we got done.
